# 04 — Penilaian Harga Wajar Saham

> Sumber: Bab IV — "Metode Sederhana untuk Menilai Harga Wajar Saham"

---

## Dua Jenis Nilai Saham

Setiap saham memiliki dua dimensi nilai yang harus dipahami:

| | Present Value | Future Value |
|--|---------------|--------------|
| Definisi | Nilai aset bersih perusahaan hari ini | Estimasi nilai saham di masa depan |
| Nama lain | Book Value | Nilai pertumbuhan |
| Cara hitung | Ekuitas ÷ Jumlah saham beredar | Present Value × (1 + ROE)^n |
| Sumber data | Laporan keuangan terbaru | Track record ROE historis |
| Ada di semua saham? | ✅ Ya | ❌ Hanya saham yang labanya konsisten |

---

## LANGKAH 1 — Hitung Present Value (Book Value per Saham)

```
Book Value per Saham = Ekuitas ÷ Jumlah Saham Beredar
```

**Yang dimaksud "Ekuitas":**
- Gunakan ekuitas yang dapat diatribusikan kepada pemilik entitas induk
- Bukan total ekuitas (yang termasuk kepentingan non-pengendali)
- Letaknya: bagian bawah neraca / laporan posisi keuangan

**Contoh ASII (Kuartal I 2014):**
- Ekuitas entitas induk: Rp88,9 triliun
- Jumlah saham beredar: 40,5 miliar lembar
- Book Value: Rp88,9 T ÷ 40,5 M = **Rp2.197 per saham**

---

## LANGKAH 2 — Hitung PBV (Price to Book Value)

```
PBV = Harga Pasar ÷ Book Value per Saham
```

**Interpretasi PBV:**

| PBV | Artinya |
|-----|---------|
| < 1 kali | Harga di bawah nilai buku → sangat murah (tapi cek fundamental) |
| 1 kali | Harga = nilai buku → wajar untuk perusahaan biasa |
| 1–2 kali | Masuk akal untuk perusahaan dengan ROE moderat |
| 2–3 kali | Wajar untuk perusahaan dengan ROE tinggi & konsisten |
| > 4–5 kali | Mahal — butuh pertumbuhan luar biasa untuk justified |

> Semakin tinggi PBV, semakin mahal. Tapi PBV tinggi bisa justified
> jika perusahaan memiliki future value yang kuat.

---

## LANGKAH 3 — Hitung ROE (Return on Equity)

ROE adalah "bunga bank" dari sebuah perusahaan — seberapa efisien
perusahaan mengembangkan modalnya setiap tahun.

```
ROE = (Laba Bersih Setahun ÷ Ekuitas) × 100%
```

**Cara setahunkan laba bersih:**

| Laporan | Rumus |
|---------|-------|
| Kuartal I | Laba × 4 |
| Kuartal II | Laba × 2 |
| Kuartal III | Laba × 4/3 |
| Kuartal IV | Tidak perlu dikali (sudah setahun penuh) |

**Interpretasi ROE:**

| ROE | Kualitas |
|-----|----------|
| < 10% | Kurang baik |
| 10–15% | Cukup |
| 15–20% | Bagus |
| > 20% | Sangat bagus |
| > 25% konsisten | Kelas premium |

**Contoh BBRI (Kuartal III 2014):**
- Laba bersih 9 bulan: Rp18,2 triliun
- Disetahunkan: Rp18,2 T × 4/3 = Rp24,2 triliun
- Ekuitas: Rp91,3 triliun
- ROE: 24,2 ÷ 91,3 × 100% = **26,5%** → sangat bagus

---

## LANGKAH 4 — Hitung Future Value

Future Value hanya bisa dihitung untuk perusahaan yang memiliki
**track record laba konsisten** — minimal 5 tahun ke belakang.

```
Future Value (tahun ke-n) = Book Value Saat Ini × (1 + ROE)^n
```

**Periode yang digunakan: 5 tahun ke depan**
- Kurang dari 5 tahun → hasilnya terlalu rendah, tidak realistis
- Lebih dari 5 tahun → terlalu spekulatif

**Contoh ASII:**
- Book Value akhir 2013: Rp2.073 per saham
- ROE asumsi: 21,2% (berdasarkan track record)

```
Tahun 2014: 2.073 × 1.212 = 2.512
Tahun 2015: 2.512 × 1.212 = 3.045
Tahun 2016: 3.045 × 1.212 = 3.691
Tahun 2017: 3.691 × 1.212 = 4.473
Tahun 2018: 4.473 × 1.212 = 5.421
```

**Future Value ASII = ~Rp5.422 per saham (5 tahun ke depan)**

---

## LANGKAH 5 — Tentukan Harga Wajar Beli

```
Harga wajar beli = jauh di bawah Future Value yang dihitung
```

> Future Value adalah perkiraan, bukan kepastian.
> Kamu tidak bisa melihat masa depan — tidak ada yang bisa.
> Yang penting: beli di harga yang **jauh lebih rendah** dari perkiraanmu,
> sehingga meski perhitunganmu sedikit meleset, kamu tetap untung.

**Contoh ASII:**
- Future Value ~Rp5.422
- Harga wajar beli: Rp6.000 atau di bawahnya
- Ideal: Rp5.500 atau Rp5.000 (makin rendah makin baik)

---

## Catatan Penting: Future Value Bersifat Subjektif

Tidak ada satu jawaban yang benar untuk future value.
Dua investor bisa punya hasil berbeda karena:
- Asumsi ROE yang berbeda
- Periode waktu yang berbeda

Teguh Hidayat menganalogikan ini dengan menebak tinggi badan seseorang —
tidak ada yang bisa tepat persis, tapi semua orang tahu apakah seseorang
itu tinggi atau pendek. Demikian pula dengan valuasi saham.

---

## Panduan Cepat: Kapan Saham Bisa Dikatakan Murah?

```
Saham MURAH jika:
  Harga pasar << Future Value (untuk saham dengan laba konsisten)
  ATAU
  PBV < 1 (untuk saham tanpa future value / perusahaan baru)

Saham MAHAL jika:
  Harga pasar > Future Value
  ATAU
  PBV > 4-5 kali tanpa justifikasi pertumbuhan yang kuat
```
