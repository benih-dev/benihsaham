## Cara BIMA Menggunakan Dokumen Ini

- Sebelum membuat keputusan investasi → baca `03`, `04`, `05`, `07`
- Sebelum membuat konten edukasi → baca dokumen yang relevan dengan topik
- Saat market crash atau IHSG turun → baca `02` dan `08`
- Saat tidak ada saham yang layak beli → baca `07` bagian kapan hold/tunggu
- Saat menyusun portofolio → baca `06`
