# Portfolio Engine Workspace

Folder ini adalah pusat kerja Portfolio Engine BenihSaham.

## Tujuan

- Menjaga portofolio riil Rp8 juta tetap bergerak dalam koridor value investing.
- Mencatat seluruh keputusan investasi secara jujur dan bisa diaudit.
- Menjadikan keputusan investasi sebagai bahan konten dan pelajaran.

## Struktur

- `daily/` untuk catatan cek pasar harian.
- `decisions/` untuk jurnal buy, hold, wait, reject, dan sell.
- `monthly/` untuk review bulanan dan trading plan bulan berikutnya.
- `quarterly/` untuk screening laporan keuangan dan revisi thesis.
- `watchlist.md` untuk daftar saham yang dipantau.
- `valuation-template.md` untuk kerangka valuasi emiten.

## Workflow

1. Catat kondisi pasar harian.
2. Update watchlist mingguan bila ada perubahan valuasi material.
3. Gunakan template valuasi sebelum menulis keputusan.
4. Simpan keputusan final di `decisions/` dengan reasoning lengkap.
5. Ringkas hasilnya ke review bulanan dan laporan board.
