# Content Engine Workspace

Folder ini adalah pusat kerja Content Engine BenihSaham.

## Tujuan

- Menjaga ritme minimal dua konten per minggu.
- Menurunkan pengalaman portofolio nyata menjadi edukasi yang mudah dicerna.
- Membangun trust melalui konsistensi, kejujuran, dan analogi sederhana.

## Struktur

- `calendar/` untuk kalender konten dan komitmen terbit.
- `backlog/` untuk ide yang belum diproduksi.
- `drafts/articles/` untuk artikel web.
- `drafts/videos/` untuk skrip YouTube.
- `drafts/newsletters/` untuk newsletter dan Substack.

## Pilar Konten

- Metodologi value investing.
- Studi kasus analisis.
- Jurnal perjalanan portofolio riil.
- Kesalahan dan lesson learned.
- Edukasi risiko dan ekspektasi realistis.

## Rules

- Minimal dua konten per minggu.
- Default mix: satu evergreen, satu portfolio/process log.
- Hindari klaim return tanpa data aktual.
- Produk berbayar tidak boleh berisi rekomendasi saham spesifik.
