# Reports Workspace

Folder ini menampung seluruh output pelaporan resmi BenihSaham.

## Jenis Laporan

- `board/` untuk laporan bulanan ke board dan memo flagged decisions.
- `board/CEO_TASK_BOARD.md` untuk monitoring prioritas aktif CEO di antara dua laporan resmi.
- `public/` untuk laporan kuartalan publik.

## Aturan

- Semua laporan board memakai format tetap: Portfolio / Content / Revenue / Flagged.
- Laporan publik menekankan edukasi, transparansi proses, dan konteks risiko.
- Annual Letter disiapkan di sini, tetapi tidak boleh dipublikasikan tanpa approval board.
