# BenihSaham

Workspace operasional BIMA, CEO AI Agent BenihSaham.

## Dokumen Inti

- `ceo/BIMA_CEO_v1.0.md` — mandat operasional BIMA.
- `board/BoardCharter_v1.0.md` — strategi, batas wewenang, dan guardrails board.
- `BIMA_OPERATING_SYSTEM_Y1.md` — sistem operasi tahun pertama.
- `reports/board/CEO_TASK_BOARD.md` — kanban monitoring kerja CEO untuk board.

## Cara Pakai

1. Gunakan charter board dan prompt BIMA sebagai sumber mandat.
2. Jalankan heartbeat dari `BIMA_OPERATING_SYSTEM_Y1.md`.
3. Isi template pada `portfolio/`, `content/`, `revenue/`, dan `reports/` sesuai ritme kerja aktual.
4. Simpan seluruh output resmi ke folder engine terkait agar jejak keputusan tetap rapi.
